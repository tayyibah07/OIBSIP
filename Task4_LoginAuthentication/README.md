# Task 4 - Login Authentication System

## Project Overview

This project is a Login Authentication System developed as part of the Oasis Infobyte Web Development Internship Program. The application allows users to register, log in, and access a secured dashboard through a simple authentication workflow.

## Live Demo

🔗 https://tayyibah07.github.io/OIBSIP/Task4_LoginAuthentication/

## Features

* User Registration
* User Login Authentication
* Dashboard Access
* Form Validation
* Responsive User Interface
* Interactive User Experience

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

Task4_LoginAuthentication/
├── index.html

├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Register a new account.
4. Log in using the registered credentials.
5. Access the dashboard page.
