# Task 1 - Calculator

## Project Overview

This project is a responsive Calculator Web Application developed as part of the Oasis Infobyte Web Development Internship Program. The application performs basic arithmetic operations through an interactive and user-friendly interface.

## Features

* Addition, Subtraction, Multiplication, and Division
* Clear (AC) functionality
* Responsive design for desktop and mobile devices
* Modern and attractive user interface
* Real-time calculation display

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

Task1_Calculator/
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in any modern web browser.
3. Use the calculator to perform arithmetic operations.

Through this project, I gained practical experience in:

* DOM Manipulation
* Event Handling in JavaScript
* Responsive Web Design
* User Interface Development
