# Task 2 - Tribute Page

## Project Overview

This project is a Tribute Page created as part of the Oasis Infobyte Web Development Internship Program. The webpage is dedicated to Dr. A.P.J. Abdul Kalam, one of India's most respected scientists and former Presidents, highlighting his achievements, contributions, and inspiring life journey.

## Live Demo

🔗 https://tayyibah07.github.io/OIBSIP/Task2_TributePage/

## Features

* Attractive and responsive user interface
* Biography and achievements section
* Inspirational content about Dr. A.P.J. Abdul Kalam
* Image integration for enhanced visual appeal
* Clean and structured webpage layout

## Technologies Used

* HTML5
* CSS3

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in a web browser.
3. Explore the tribute page and learn about Dr. A.P.J. Abdul Kalam.
