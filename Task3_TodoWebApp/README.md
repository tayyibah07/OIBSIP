# Task 3 - To-Do Web Application

## Project Overview

This project is a To-Do Web Application developed as part of the Oasis Infobyte Web Development Internship Program. The application helps users manage daily tasks efficiently by allowing them to add, complete, and remove tasks through a simple and interactive interface.

## Live Demo

🔗 https://tayyibah07.github.io/OIBSIP/Task3_TodoWebApp/

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Interactive and user-friendly interface
* Responsive design for different screen sizes
* Dynamic task management using JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

Task3_TodoWebApp/
├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in a web browser.
3. Add, manage, and organize your tasks efficiently.
